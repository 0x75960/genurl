gURL
=====

generate .url file from url

Basic usage
------------

```sh
gurl --help
```

```txt
Usage of this:
         gurl [options] https://url.to/gen/url

  -n string
        output name (default "link")
  -o string
        output dir path (default ".")
```

Setup
-----

### require

Go

### install

```sh
go get -u github.com/0x75960/gurl
```
